📝 Regras do .htaccess

# Regras para sistema de eventos
RewriteRule ^evento/([a-z0-9-]+)/?$ /evento/index.php?slug=$1 [NC,L]
RewriteRule ^evento/checkout/([a-z0-9-]+)/?$ /evento/checkout.php?evento=$1 [NC,L]
RewriteRule ^evento/pag-pix/([a-z0-9-]+)/?$ /evento/pagamento-pix.php?evento=$1 [NC,L]
RewriteRule ^evento/pag-success/?$ /evento/pagamento-sucesso.php [NC,L]
RewriteRule ^evento/pag-success/([a-z0-9-]+)/?$ /evento/pagamento-sucesso.php?evento=$1 [NC,L]

🔄 Fluxo Completo

Jornada do usuário com URLs amigáveis:

  1. /evento/goianarh - Usuário vê o evento
  2. /evento/checkout/goianarh - Seleciona ingressos e faz checkout
  3. /evento/pag-pix/goianarh - Realiza pagamento via PIX
  4. /evento/pag-success/goianarh - Confirma pagamento com sucesso

⚠️ Observações

  • Cache: Pode ser necessário limpar o cache do navegador/servidor
  • Teste: Use modo anônimo/privado para testar
  • Logs: Verifique logs do Apache se alguma URL não funcionar
  • Ordem: As regras são processadas em ordem, por isso a mais específica vem primeiro
